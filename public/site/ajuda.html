Avisos Importantes!

Este projeto visa auxiliar o aprendizado de linguagem assembly para os alunos do curso superior de Tecnologia em Análise e Desenvolvimento de Sitemas (TADS) na disciplina de Arquitetura de Computadores.

Este projeto encontra-se em fase de desenvolvimento, portanto nem todos os recursos estão implementados.

Caso o simulador apresente algum problema, por favor, relate para Victor Carlquist ou Matheus Liberato.

Observações

  • O emulador suporta apenas números decimais;

  • A Pilha é inicializada no endereço 16(0x010).

Exemplos

O programa a seguir compara o registrador R16 com o número 4, caso o valor do R16 for maior ou igual a 4 o salto ocorrerá para a label 'PULA' e o registrador R18 não receberá o valor 55.
LDI R16,5
CPI R16,4
BRGE PULA ;verifica se R16 >= 4
LDI R18,55
PULA: LDI R20,5
O programa a seguir mostra o funcionamento da instrução CALL e o uso de subrotina.
JMP INICIO ;vai para o inicio do programa

ROTINA:
LDI R16,10
RET

INICIO: LDI R18,5
CALL ROTINA
MOV R18,R16
